# Retail Sales & Customer Insights Dashboard

Interactive Power BI dashboard for analyzing sales, customer behavior, and product performance.

## Key Features
- Sales trend visualization
- Product performance & categories
- Customer repeat behavior
- Revenue by channel (in-store vs delivery)
- Metrics: AOV, CAC (simulated), repeat rate, churn estimate

## Tools
- Power BI
- Excel / Python (pandas)

## Structure
- data/: raw & cleaned CSV files
- dashboards/: .pbix file
- docs/: project notes & case study
